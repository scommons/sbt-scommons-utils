## sbt-scommons-plugin
Sbt auto-plugins with common tasks/utils for Scala Commons modules

### How to add it to your project

```scala
// inside plugins.sbt

addSbtPlugin("org.scommons.sbt" % "sbt-scommons-plugin" % [version])
```
